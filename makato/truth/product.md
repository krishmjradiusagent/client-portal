# Product Truth — Radius Agent

## Product

Radius Agent is a real estate technology platform for agents, teams, transaction coordinators, and admins.

Primary product areas:
- CRM
- Clients
- Transactions
- Offers
- Listings
- Auditing
- CDA
- Envelopes
- Signing flows
- Messaging
- Vendors
- Collaborators
- AI assistance

## Platforms

### Web
Use for:
- CRM operations
- transaction management
- auditing
- team settings
- envelope/document flows
- dense tables and side panels

Default:
- light mode
- compact professional layouts
- shadcn/ui + Tailwind

### Mobile
Use for:
- agent workflows
- client workflows
- quick actions
- messaging
- showing status and progress

Default:
- dark mode first
- bottom sheets
- cards
- sticky CTAs

## Product Tone

Quiet premium.
Professional.
Clear.
Operational.
Real estate specific.

Avoid:
- vague SaaS copy
- oversized hero sections
- decorative UI
- random colors
- dead-end flows

## Product Principles

1. Role-aware UX.
2. Every flow needs states.
3. Every critical action needs clear next step.
4. Preserve existing Radius patterns.
5. Use real estate language.
6. Make complex transaction workflows feel controlled.
