# Design System Truth

## Direction

Quiet premium product UI.

Inspired by:
- Airbnb clarity
- Zillow familiarity
- modern shadcn dashboards
- dense CRM systems

## Web Defaults

- shadcn/ui components
- Tailwind semantic tokens
- dense tables
- side panels
- cards for grouped info
- dialogs for focused actions
- badges for status
- command/search for selector-heavy flows

## Mobile Defaults

- dark mode first
- bottom sheets
- card-based hierarchy
- sticky primary CTA
- compact labels
- thumb-safe actions

## Typography

- Heading max: 24px unless task explicitly says otherwise
- Prefer semibold headings
- Body text: 14px default
- Caption/meta: 12px
- Avoid large marketing typography in product UI

## Spacing

Use:
- 4px micro
- 8px tight
- 12px compact
- 16px standard
- 24px section

Do not invent random spacing.

## Accessibility

Required:
- WCAG-AA contrast
- visible focus states
- keyboard support
- semantic labels
- modal focus trap
- no color-only meaning
- validation messages tied to fields
