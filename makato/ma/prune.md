# MA — Prune Context

MA means remove noise before asking an LLM to work.

## Keep

- Current task
- Relevant workspace path
- Relevant Radius role
- Relevant platform
- Relevant Figma/screenshot facts
- Relevant decisions
- Relevant design-system rules
- Latest handoff
- Kaizen rules

## Ignore

- Unrelated Radius modules
- Old experiments
- Raw transcripts
- Unused screenshots
- Generic UI advice
- Non-current tasks
- Past flows not referenced by current task
