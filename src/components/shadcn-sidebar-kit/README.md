# shadcn Sidebar Kit

Drop-in sidebar shell using official shadcn/ui Sidebar primitives.

## Install

```bash
npx shadcn@latest add sidebar avatar dropdown-menu collapsible separator
```

## Files

- `components/app-sidebar.tsx` — sidebar component
- `components/app-shell.tsx` — layout wrapper
- `app/layout-example.tsx` — Next.js example
- `docs/CODEX_LINK_PROMPT.md` — prompt to make Codex wire it safely

## Rule

Use shadcn tokens only. No custom visual system.
